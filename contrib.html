There used to be an extended version of 2.19, "sxvi", called xvi-sjis on freshports.org, which handles Japanese SJIS. Its source code is here and on archive.org

There are a couple of binary file editors called XVI that are not xvi:

  • The one mentioned at Sven Guckes "vi pages" is B. Sartirana's screen oriented hexidecimal/octal editor from 1985, whose commands are a subset of those of "vi".
  • XVI32 is a freeware hex editor running under Windows 95 to XP. The name is a pun on XVI being 16 in roman numerals.
